## Security Policy

If you believe you have found a vulnerability in Crosscheck, please report it
privately to:

- security@crosscheck.news

Please include:

- a short description of the issue
- the affected URL, route, or component
- reproduction steps or a proof of concept
- the impact you believe it has

We prefer private disclosure so we can investigate and ship a fix before public
discussion. We aim to acknowledge reports within 2 business days.

For the live machine-readable disclosure contact, see:

- `/.well-known/security.txt`

Operational security notes and the current app-level baseline live in:

- `docs/security.md`
